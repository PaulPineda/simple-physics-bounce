# Barebones Boilerplate

Installs the following as devDepandancies:
- webpack
- webpack-dev-server

- babel-core
- babel-loader
- babel-preset-es2015
- babel-preset-stage-2

- mocha
- chai

- eslint
- eslint-plugin-html

Creates the following folder structure:
|- .git
|- /node\_modules/
|- /src/
|  |- index.js
|- .babelrc
|- /eslintignore
|- .eslintrc
|- favicon.ico
|- README.md
|- index.html
|- package.json
|- webpack.config.babel.js
